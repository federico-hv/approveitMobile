nativecontrols Module

Description

This module is used to create native controls in titanium which are not available in titanium APIs.

Accessing the nativecontrols Module

To access this module from JavaScript, you would do the following:

var nativecontrols = require("learnappcelerator.nativecontrols");

The nativecontrols variable is a reference to the Module object.

Reference

1.createRadioGroup();

This method used to create the RadioGroup with radio buttons

Properties:

a.) textColor - specify the Radio button text eg. #FF0000 (6 digit only) default text color is #000000

b.) selectedIndex - specify the default selected radio button index. eg: selectedIndex : 2. default value is 0.

c.) layoutType - specify the radio group layout orientation "vertical" or "horizontal". default layoutType is "horizontal" d.) buttons - Array of objects with id and text value. The buttons created with text. eg, buttons : [{ id : 1, text : 'value 11' }, { id : 2, text : 'value 12' }, { id : 3, text : 'value 13' }]

Usage

var radiogroup = nativecontrols.createRadioGroup({ width : Ti.UI.SIZE, height : Ti.UI.SIZE, top : 100, left : 10, textColor : "", selectedIndex : 2, layoutType : "vertical", buttons : [{ id : 1, text : 'value 11' }, { id : 2, text : 'value 12' }, { id : 3, text : 'value 13' }] });
